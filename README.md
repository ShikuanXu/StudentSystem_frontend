Student Management System (Frontend)

This is the frontend of a full-stack student management system.
The project is built with React and communicates with a Spring Boot backend.

 Note: This project is still under development. Some backend features are not fully completed yet.

 Tech Stack
React
Vite
Axios
 Current Features
Display student list (from backend API)
Integration with backend REST APIs
Error handling for API requests
 Current Limitations
Some backend features are incomplete
Not all endpoints are fully connected
UI design is basic and still being improved
No authentication system yet

 How to Run
Clone the repository
Install dependencies:
npm install
Start the development server:
npm run dev
Open in browser:
http://localhost:5173

 Make sure the backend server is running on:

http://localhost:8080
